# Telegram-бот с искусственным интеллектом для поиска заведений и погоды

Telegram-бот, созданный с использованием **aiogram** и **FastAPI**, который предоставляет:

* 🍽️ **Поиск ресторанов и кафе поблизости** (по геолокации пользователя)
* 🌤️ **Показ погоды** по текущему местонахождению
* 🤖 **Поиск информации** по пользовательскому тексту
* 😊 **Анализ эмоций** по пользовательскому тексту

Бот определяет ваше местоположение, находит ближайшие заведения питания, показывает актуальную погоду, поиск информации и анализирует эмоциональный тон сообщений.

---

## Обзор проекта

Этот проект объединяет в одной системе взаимодействие с Telegram-ботом, обработку данных через бэкэнд API, геолокацию, поиск заведений питания, прогноз погоды и анализ настроения.

**Основные сценарии использования:**

* 📍 Отправка геолокации и получение списка ресторанов и кафе
* 🌦️ Запрос погоды по текущему местоположению
* 💬 Отправка текста для анализа эмоций
* 📱 Просмотр отформатированных результатов с подробной информацией о заведениях

---

## Функции

### 1. Поиск ресторанов и кафе

* Принимает геолокацию от пользователя в Telegram
* Использует **GIS API** для поиска заведений питания поблизости
* Возвращает список ресторанов и кафе с информацией:
  * Название заведения
  * Расстояние от пользователя

### 2. Показ погоды

* Определяет местоположение пользователя
* Получает актуальные данные о погоде через **WEATHER API**
* Возвращает пользователю:
  * Текущую температуру
  * Тип погоды (ясно, облачно, дождь и т.д.)

### 3. Анализ эмоций

* Принимает текстовый ввод от пользователя
* Классифицирует эмоциональный тон сообщения
* Определяет:
  * Основную эмоцию (Позитивный / Негативный)
    
### 4. Поиск информации

* Принимает текстовый ввод от пользователя
* Обрабатывает запрос пользователя
* Возвращает пользователю:
  * Информацию
  
В проекте используется **FastAPI** в качестве бэкэнд-сервиса для:

* Обрабатывать запросы от Telegram-бота
* Запросы на поиск заведений по геолокации
* Обрабатывать запросы на получение погоды
* Обрабатывать запросы на анализ эмоций
* Проверять входные данные и ответы

## Используемые API

### 1. GIS API

Используется для поиска ресторанов, кафе и других заведений питания по местоположению.

**Назначение API:**

* Поиск заведений поблизости от координат
* Фильтрация по типу (ресторан, кафе, бар и т.д.)
* Получение детальной информации:
  * Название
  * Адрес

### 2. WEATHER API

Используется для получения актуальных данных о погоде.

**Назначение API:**

* Получение текущей погоды по координатам
* Детальная информация:
  * Температура
  * Облачность

### 3. DeepSeek API

Используется для поиска информации.

**Назначение API:**

* Анализ и обработка текстовых данных
* Поиск информации
